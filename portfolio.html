Sport · Communauté En prod depuis 2000+

Tennis-Contact — Plateforme communautaire tennis

Le problème : Des joueurs locaux voulaient organiser des parties sans dépendre d'apps génériques. L'outil existant tournait sur un legacy PHP vieillissant, difficile à maintenir.

Ce que j'ai fait : Migration complète du legacy vers Symfony + React moderne. Refonte de la gestion clubs/coaches, de l'organisation des parties et du panel d'administration. Docker pour le déploiement, MySQL pour la persistance.

  • ✓ Migration PHP legacy → Symfony sans perte de données ni downtime
  • ✓ Visibilité locale améliorée, pratique du sport facilitée
  • ✓ Revenus via coaches et partenaires, actif depuis plus de 20 ans
PHP / Symfony React MySQL Docker Migration legacy
Freelance · Outil métier En prod

FS-Web-Conseil — CRM et site vitrine sur-mesure

Le problème : Gérer une activité freelance avec Excel et des outils génériques (facturation, suivi clients, temps) coûte plus en friction qu'en économies réalisées.

Ce que j'ai fait : Monorepo avec site vitrine Next.js et CRM Symfony custom conçu avec les patterns DDD/CQRS — gestion des clients, des devis, du temps et de la facturation. Tout dans un seul outil, adapté au process réel.

  • ✓ Excel remplacé par un outil adapté au vrai flux de travail
  • ✓ Gain de productivité mesurable sur les tâches admin
  • ✓ Démonstration concrète des patterns DDD/CQRS en production
Next.js Symfony React PostgreSQL DDD / CQRS Docker
E-commerce · Automatisation En prod

Sellermania/MVS — Connecteur e-commerce automatisé

Le problème : Un e-commerçant synchronisait manuellement commandes, stocks et suivis entre sa marketplace (Sellermania) et son logisticien (MVS). 20h/semaine perdues, risque d'erreurs constant.

Ce que j'ai fait : Connecteur API avec 3 flux cron automatisés : synchronisation des commandes, mise à jour des stocks, suivi des expéditions. Déduplication intégrée pour éviter les doublons entre systèmes.

  • ✓ Élimination des tâches de synchronisation manuelle
  • ✓ Réduction des erreurs de commandes
  • ✓ Scaling des ventes sans surcharge administrative
PHP MySQL API Sellermania API MVS Docker Cron
